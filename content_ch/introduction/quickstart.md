# 主控器安装

安装注意事项：
-   水平地安装在飞行器面板上并尽量安装在飞行器的正中心，避免震动引起飞行器飞行过程中不稳定。
-   主控制器安装时，必须将印有V9标识的一面向上，安装方向可分为：前向、左向、后向和右向四个方向安装；默认为前向安装，小三角标志尖角朝向应与飞行器机头方向一致。如设置右向安装，小箭头应指向飞行器右侧

# GPS安装

安装注意事项：

-   安装GPS模块时，必须将印有标记的一面朝上，小三角标志尖角朝向与飞行器机头方向一致。
-   GPS模块为磁性敏感设备，安装和使用应远离其它电子设备和强磁性物质，架高机体平面10cm以上。

# 双天线RTK安装

-   双天线测向RTK模块的两个天线必须是正前后（1号后，2号前）、正左右（1号左2号右）安装，并且在APP上对应设置正确。模块标有ANT1的为1号天线，标有ANT2的为2号天线。
-   两个天线之间的距离大于40cm

<div  align="center"> <img src="pictures/RTK安装.png" width="50%" height="50%">
</div>

__D3-H灯语:__
- RX-1：串口1收数据状态灯、绿灯（出线口），接收一包数据闪烁一次；
- RX-2：串口2收数据状态灯、蓝灯（插座口），接收一包数据闪烁一次；
- ANT-1：主天线故障灯，正常时灭灯，蓝灯亮时天线断路故障，红灯亮时天线短路故障；
- ANT-2：从天线故障灯，正常时灭灯，蓝灯亮时天线断路故障，红灯亮时天线短路故障。

# 雷达安装

-   仿地雷达水平安装在飞机底部，雷达下部不要有任何遮挡
-   避障雷达安装在飞机的前部/后部，角度上仰12度。参照示意图
